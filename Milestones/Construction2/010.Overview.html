Overview

This section give an overview of the contents of the Construction 2 document

The documentation completed during the Construction 2 is to help the team finalize construction of the project and to work towards a final release. A successful Construction 2 document will include the following:

  • Completed users guide
  • Completed use case models
  • Completed Glossory
  • Completed system architecure
  • Completed UML diagrams
  • Working applciation code
  • Test code
  • Contribution summary

At the end of the Construction 2 we should have the majority of the project completed and ready for release.
